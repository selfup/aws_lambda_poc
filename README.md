# AWS Lambda Node.js Starter Kit

Get started after clone: `npm i`

Now code something awesome :tada:

## Packaging

`./scripts/build_and_zip.sh`

Now upload the zip or use the [aws-cli](https://aws.amazon.com/cli/) :rocket:

That's it :joy:
